# Once-javiera
Clase de media tecnica.
## Pasos para crear un repositorio
* En tu navegador de preferencia busca GitHub y inicia sesion con tu cuenta.
* Dirigete a la seccion de repositorios una vez dentro en la parte superior te debe de aparecer un boton verde que dice "New" o "New repositorio" presionalo.
* Una vez dentro tendras que darle un nombre al repositorio y en la seccion de visible: tendras que ponerlo en publico.
